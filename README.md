# TensorFlow

This repository contains a collection of deep learning experiments and model implementations using TensorFlow. Each notebook explores core concepts or techniques in machine learning and deep learning.

- [**Linear Regression**](Linear_Regression.ipynb) : A basic implementation of linear regression to predict housing prices using the Boston Housing dataset and TensorFlow.
- [**Classification with MLP**](Classification_MLP.ipynb) : MNIST digit classification using a Multi-Layer Perceptron (MLP) model.
- [**Convolutional Neural Networks (CNNs)**](CNN.ipynb) : Implementing a CNN for CIFAR-10 image classification.
- [**Pre-trained Models**](Pre_Trained_Models.ipynb) : Image classification using pre-trained ImageNet models from TensorFlow Hub.
- [**Transfer Learning and Fine-Tuning**](Fine_Tuning.ipynb) : Transfer learning and fine-tuning a VGG16 model to perform classification on the GTSRB dataset.
- [**Semantic Segmentation**](Semantic_Segmentation.ipynb) : Semantic segmentation using HRNet models from TensorFlow Hub, trained on the CamVid dataset.
- [**Object Detection**](Object_Detection.ipynb) : Object detection using EfficientDet models from TensorFlow Hub.
